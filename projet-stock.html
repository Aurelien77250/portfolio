Projet de développement d'une application de Gestion de stock

Dans le cadre du cours de développement du BTS SIO, nous avons réalisé une application de gestion de stock. Ce projet comprenait la mise en place d’un système d’inscription et de connexion sécurisé, permettant aux utilisateurs d’accéder à l’application selon leur profil. L’objectif était de gérer les produits en stock, avec des fonctionnalités comme l’ajout, la modification et la suppression d’articles, tout en assurant une interface simple et fonctionnelle.

Prérequis :
Python
VS Code
Base de données

Procédure :

1) Préparer l’environnement de travail
  • Installer Python
  • Créer un nouveau dossier de projet
2) Définir la structure du projet
  • Créer les fichiers suivants : Stock.py / Inscription / Connexion
  • Création de la base de données "stock"
3) Créer le système d’inscription/connexion
  • Permettre à un utilisateur de s’inscrire (nom d’utilisateur + mot de passe)
  • Enregistrer les utilisateurs dans une base de données
  • Permettre la connexion en vérifiant les identifiants
4) Créer les fonctions de gestion de stock (stock.py)
  • Ajouter un produit (nom, quantité, prix)
  • Modifier ou supprimer un produit
  • Lister tous les produits en stock
5) Distinction en fonction des utilisateurs
  • Demander à l’utilisateur de se connecter ou s’inscrire
  • Une fois connecté, afficher un menu avec les actions disponibles : ajouter, modifier, supprimer, afficher les produits
  • Sauvegarder les modifications dans le fichier ou la base de données
6) Ajouter des améliorations
  • Interface graphique avec Tkinter
  • Export des données en CSV
